## Hello readers 👋

If you want to understand how memory works under the hood and build your own custom memory allocator from scratch, you're in the right place.

This repository is not just about code — it's a **guided journey**.  
You'll first learn the fundamental concepts required to understand memory management, and then apply that knowledge step-by-step to build a working allocator.

---

> By the end of this guide, you won’t just use `malloc` — you’ll understand how to build it.
## What you'll learn

- How memory is managed in programs
- Stack vs Heap
- How the heap grows (`sbrk`)
- How allocators manage memory
- How to build your own `malloc` and `free`
